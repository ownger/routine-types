# TypeScript Utility Snippets

A curated collection of practical TypeScript utility types and helper functions for everyday development. All examples are production-ready and fully type-safe.

## Overview

This project provides a comprehensive set of TypeScript utility types that extend the standard library. Each snippet includes detailed documentation, usage examples, and implementation details. The collection is designed for developers who need reliable, battle-tested type utilities without external dependencies.

## Features

- Production-ready utility types validated for real-world use cases
- Zero dependencies - pure TypeScript implementations
- Progressive Web App with offline support
- Responsive design with light/dark theme support
- Syntax highlighting for enhanced code readability

## Technology Stack

- Rspack (high-performance Rust-based bundler)
- Prism.js for syntax highlighting
- Semantic HTML5
- Modern CSS with custom properties
- ES6+ JavaScript modules
- Service Worker for offline functionality

## Quick Start

```bash
git clone https://github.com/ownger/routine-types.git
cd routine-types
npm install
npm run dev
```
